GitHub 云端自动合并（零部署 · 推荐）

用你的 GitHub 仓库当云端：面试官提交评分将自动写回仓库文件，HR 在结果页点「☁ 从云端刷新」即汇总，无需来回传链接。
Token 建议用细粒度令牌（仅勾选本仓库 Contents: Read/Write），会写进发给面试官的链接里，用后可在 GitHub 撤销。留空则退化为本地 + 离线回传链接模式。

发给面试官 / 多人协作

云端模式：点「复制面试官链接」发面试官 → 其打开后仅见评分页，提交即自动上传合并。
离线模式（未设云端）：面试官打开链接评分，点「复制我的评分链接」回传 → HR 打开即合并。
链接过长可改用「导出/导入」JSON 文件经微信发送合并。

STAR 评分指引

A 行动：你独立落地的具体动作？  R 结果：最终量化成果？
评分基于回答的具体性、真实性、量化程度，避免光环效应。
